# Product Notes

The product is a Java-first AI AgentOps workbench for multi-agent R&D collaboration.

The first golden scenario is a banking financial-technology scenario:

```text
Bank credit-card installment campaign configuration and approval system R&D collaboration
```

The workbench should demonstrate:

- intent analysis
- agent recommendation
- DAG orchestration
- HumanGate approval
- artifact generation
- REFLECTION
- Lessons
- auditability
- event replay

The system must avoid real bank customer data, real internal system names, and sensitive risk strategy details.
